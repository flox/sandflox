    **Step 1: Create env.go**

    Create `env.go` in package main with:

    1. `defaultAllowlist` var ([]string): Exact var names that always pass through:
       ```
       "HOME", "USER", "LOGNAME", "SHELL", "TERM", "PATH",
       "LANG", "LANGUAGE",
       "COLORTERM", "TERM_PROGRAM", "TERM_PROGRAM_VERSION",
       "TMPDIR",
       "SANDFLOX_ENABLED", "SANDFLOX_MODE", "SANDFLOX_PROFILE",
       ```

    2. `allowedPrefixes` var ([]string): Prefix patterns that pass through:
       ```
       "FLOX_", "NIX_", "__", "LC_", "XPC_",
       ```

    3. `blockedPrefixes` var ([]string): Always blocked, even if matching an allowed prefix:
       ```
       "AWS_", "AZURE_", "GCP_", "GCLOUD_",
       "SSH_", "GPG_",
       "DOCKER_", "KUBE",
       "OPENAI_", "ANTHROPIC_", "MISTRAL_",
       "GITHUB_", "GITLAB_", "BITBUCKET_",
       "HOMEBREW_",
       "NPM_", "YARN_", "CARGO_",
       "DATABASE_", "DB_", "REDIS_", "MONGO",
       "STRIPE_", "TWILIO_", "SENDGRID_",
       "SLACK_", "DISCORD_",
       ```

    4. `blockedExact` var ([]string): Exact names always blocked:
       ```
       "GITHUB_TOKEN", "GH_TOKEN", "GITLAB_TOKEN",
       "HF_TOKEN", "HUGGING_FACE_HUB_TOKEN",
       "OPENAI_API_KEY", "ANTHROPIC_API_KEY",
       "SECRET_KEY", "API_KEY", "API_SECRET",
       "PASSWORD", "PASSWD",
       ```

    5. `forcedVars` var (map[string]string): Always set, overriding parent:
       ```
       "PYTHONDONTWRITEBYTECODE": "1",
       "PYTHON_NOPIP": "1",
       ```

    6. Helper `envToMap(env []string) map[string]string` -- uses `strings.SplitN(entry, "=", 2)` per Pitfall 3. Returns map of KEY->VALUE.

    7. Helper `isBlocked(key string) bool` -- returns true if key matches any `blockedPrefixes` prefix OR is in `blockedExact`. Uses `strings.HasPrefix` for prefix matching.

    8. `BuildSanitizedEnv(cfg *ResolvedConfig) []string`:
       - Call `os.Environ()` and convert to map via `envToMap`
       - Build `seen` map[string]bool to track already-added keys (dedup)
       - Phase 1: iterate `defaultAllowlist`, add matching vars from envMap (skip if isBlocked)
       - Phase 2: iterate `allowedPrefixes`, for each prefix scan envMap for matching keys not already seen and not isBlocked
       - Phase 3: iterate `cfg.EnvPassthrough`, add matching vars from envMap (these bypass isBlocked -- user explicitly chose them)
       - Phase 4: apply `forcedVars` (overwrite any existing value for those keys)
       - Sort the result slice with `sort.Strings` per Pitfall 5
       - Return the sorted slice

    Imports: `"os"`, `"sort"`, `"strings"`.

    Use section separator comments matching codebase style:
